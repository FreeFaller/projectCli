# 通用项目结构

### 环境准备

-   [Node.js](https://nodejs.org/en/) 
-   [Yarn](https://yarnpkg.com/zh-Hans/docs/install)，安装最新版即可

进入项目，安装依赖，执行：

```shell
➜ git:(dev) yarn
```

执行完毕，通过命令，启动项目：

```shell
➜ git:(dev) yarn dev
```

### 启动的端口
PORT=8082

### 启用 mock 数据
执行：

```shell
➜ git:(dev) yarn amock
```

mock 数据在 http://localhost:8002 可以访问
